# 사이트 라우터 구조

## 1. 사이트 맵

- / : Home
- /signin : 로그인 페이지
- /signup : 회원가입 페이지
- /forget-password : 비밀번호 찾기
- /reset-password : 비밀번호 재설정
- /profile/아이디 : 사용자 프로필 페이지
- /post : 전체 게시글
- /post/아이디 : 상세 게시글
